# Obsługa plikow i JSON

Czytanie, zapis, parsowanie plikow TXT i JSON. Wzorce interview.

## Zawartosc

| Plik | Opis |
|------|------|
| `sciaga_interview_json_txt.py` | Gotowe wzorce rekrutacyjne: JSON load/update, notatnik z timestamp |

### cwiczenia/

| Plik | Opis |
|------|------|
| `funkcje_pliki_json.py` | 10 ćwiczeń: zapis/odczyt TXT, JSON, generator haseł, filter/lambda |
| `funkcje_pliki_json_2.py` | 10 ćwiczeń: notatki, input parsing, random, kalkulator z historia |
| `funkcje_pliki_json_3.py` | 5 zaawansowanych: notatnik z data, config JSON, CSV stats, merge plikow |
| `cwiczenia_menu.py` | Interaktywne menu z 10 cwiczeniami do uruchomienia |

### pliki_przykladowe/

Przykładowe pliki danych używane w cwiczeniach (JSON, TXT).

## Uwaga

Ćwiczenia łączą także **funkcje** i **obsługę wyjatkow** - to nie tylko pliki,
ale kompleksowe zadania integrujace wiele umiejetnosci.
