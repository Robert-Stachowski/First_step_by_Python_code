## Kluczowe pojęcia

- `def` - definiowanie funkcji
- `lambda` - funkcje anonimowe (jednolinijkowe)
- `map(func, iterable)` - zastosuj funkcje do każdego elementu
- `filter(func, iterable)` - filtruj elementy
- `reduce(func, iterable)` - zredukuj do jednej wartości
- `any()` / `all()` - czy cokolwiek / czy wszystko jest True
