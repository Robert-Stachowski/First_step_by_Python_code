# Podstawy Pythona

Fundamenty języka: zmienne, typy danych, kolekcje, petle, slicing, comprehension.

## Zawartość

- `hello.py` - punkt startowy, "Hello, World!"
- `dobre_praktyki.py` - checklista dobrych praktyk w Pythonie

### zmienne_i_typy_danych/

| Plik | Opis |
|------|------|
| `typy_danych_metody.py` | Kompletna ściąga metod dla list, set, dict, tuple, string |
| `nawiasy_w_pythonie.py` | Przewodnik po nawiasach `()`, `[]`, `{}` w Pythonie |
| `wywolywanie_skladnia.py` | Kiedy używać `()`, `[]` i `.` - prosta ściąga |
| `cwiczenia_podstawy.py` | Ćwiczenia z input, arytmetyki, petli, list |
| `cwiczenia_kolekcje.py` | Ćwiczenia z list, setow, słowników, krotek |

### slicing_i_comprehension/

| Plik | Opis |
|------|------|
| `slicing_comprehension.py` | Kompletny przewodnik po slicingu + list/dict/set comprehension |
| `cwiczenia_comprehension.py` | Ćwiczenia praktyczne z comprehension i Fibonacci |

## Kolejnosc nauki

1. `hello.py` - zacznij tutaj

3. `slicing_i_comprehension/` - opanuj slicing i comprehension
4. `dobre_praktyki.py` - przeczytaj przed pisaniem większego kodu
