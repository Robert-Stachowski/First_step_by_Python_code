# Projekty Portfolio

Gotowe, pelne projekty demonstrujace umiejetnosci praktyczne.
Kazdy projekt ma wlasne README z opisem, instrukcja uruchomienia i struktura.

## Spis projektow

### Weather_CLI

Profesjonalny klient API pogodowego.
- requests.Session z timeout i walidacja
- argparse z kodami wyjscia (0/1/2)
- Pelne testy z mock (pytest)
- Architektura: separacja klienta od CLI

### Kalkulator

Kalkulator OOP z historia operacji.
- Mapowanie operacji przez slownik
- Zapis historii do pliku
- Obsluga bledow (dzielenie przez zero)
- Zasada SRP (Single Responsibility Principle)

### Mini_Explorer_CLI

Eksplorator systemu plikow w terminalu.
- pathlib zamiast os.path
- Flagi: --count, --list, --ext, --info
- argparse z subkomendami
- Obsluga bledow i edge cases

### Randfacts

Prosty generator losowych faktow.
- Uzycie zewnetrznej biblioteki (randfacts)
- Petla interakcji z uzytkownikiem
- Minimalna, ale kompletna aplikacja
